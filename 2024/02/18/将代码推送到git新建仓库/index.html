将代码推送到git新建仓库

有时候需要将一个代码放到一个新的git仓库，然后方便后续版本管理。

  1. 首先去远程git新建仓库
  2. 然后本地git初始化
  3. git add .
  4. git commit -m 'xxx'
  5. git remote add origin 新建仓库地址
  6. git pull origin master 这步是因为远程新建仓库和新建的仓库内容冲突
  7. git push origin -u master

之后的推送就是常规的步骤：

git add .

git commit -m 'xxx'

git push origin master
